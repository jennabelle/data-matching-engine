# Data Matching Engine

Takes your CSV file, parses it, and returns a list of same records that exist in database.

###Getting Started###

    npm install

###To Start Server###

    npm run dev

Go to:

    http://localhost:8080/webpack-dev-server

###To Run Unit Tests###

    npm run test
