# Data Matching Engine

Takes your CSV file, parses it, and returns a list of same records that exist in database.

###Getting Started###

    npm install

###To Start Server###

    npm start

Go to:

    http://localhost:8080/api/setupAccounts

###To Run Unit Tests###

    npm run test

###Notes on Process###



###Tech Stack###

    Node.js
    ES6
    Babel
    Mongoose.js
    MongoDB
    Express
    Webpack
